### node开发包的项目规范

```bash
    .
    ├── build/ # 构建工具目录
    ├── dist/ # 构建的输出目录
    ├── docs/ # 文档说明目录
    ├── node_modules/ # 相关的依赖文件目录
    ├── src/ # 源文件目录
    ├── test/ # 测试文件目录
    ├── .babelrc # babel配置文件
    ├── .eslintrc.json # lint配置文件
    ├── .gitignore # git提交的忽略文件
    ├── index.js # 入口文件，这里是index.js，你可以自己定义一个
    ├── LICENSE #  许可证， 会在后期有文章介绍不同的许可证的差异
    ├── package.json # 依赖包的文件，会在后期有篇文章介绍
    │
    └── README.md # 说明文件，必须
```
